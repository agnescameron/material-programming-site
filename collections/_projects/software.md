For the first series of [Material Programming Workshops](/workshops), we made a [forked version](https://agnescameron.github.io/knitout-live-visualizer/) of the CMU Knitout Visualiser tool to provide a visualisation interface that would allow students to write knitting code using the knitout-frontend tools, add common features like a waste section, and export directly to the kcode format.

For more information on how to use this tool, please see the [guide](https://soup.agnescameron.info/2026/04/09/cheatsheet.html) here, and for more information about the design process, there is a write-up [here](https://soup.agnescameron.info/2026/04/01/transfers.html).
