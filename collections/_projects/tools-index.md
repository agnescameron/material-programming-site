Since the start of this project, we have been collecting a [collaborative index](https://docs.google.com/spreadsheets/d/1Mk6qIkn9i-3fB2CGQtpkl0AzOCBJlxThXYyPawyuev8/edit?gid=0#gid=0) of open-source knit design software. The aim of this is to both collate resources for different practicioners to use, and to map the landscape of knit tools.

The index is openly-editable, contributions are welcome!
